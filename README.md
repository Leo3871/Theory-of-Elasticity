 Деформация упругого тела в поле скоростей

Проект по теории упругости: моделирование движения и деформации квадратного тела в заданном поле скоростей.

 Описание
Программа рассчитывает траектории материальных точек квадратного тела в поле скоростей:
- v₁ = sin(t) * x₁
- v₂ = t * x₂

Визуализируются:
- Траектории движения точек
- Линии тока
- Деформация тела во времени
- Векторные поля скоростей
